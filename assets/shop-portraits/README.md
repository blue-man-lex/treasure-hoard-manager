```
shop-portraits/
├── weaponsmith/     # Кузнец
├── armorer/         # Бронник  
├── alchemist/        # Знахарь
```

## Использование

1. Добавьте портреты в соответствующие папки
2. Система автоматически определит тип магазина по выбранным категориям
3. При создании магазина установится правильная иконка и название

## Адаптивность

Система поддерживает адаптивное изменение ассортимента в зависимости от уровня игроков:
- **Множитель редкости:** настраивается % за каждый уровень
- **Поэтапное развитие:** разные группы редкости активируются на разных уровнях
- **Автообновление:** при входе игроков и изменении уровня
