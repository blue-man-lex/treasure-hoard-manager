# 🎮 **ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ LOOT PILE**

## 🚀 **Быстрый старт**

### **Для GM:**
1. **Откройте компендиум** с предметами (например, D&D Basic Rules)
2. **Найдите предмет** который хотите добавить на сцену
3. **Перетащите предмет** на пустое место на сцене
4. **Готово!** Лутбокс автоматически создастся с предметом внутри

### **Для игроков:**
1. **Откройте инвентарь** своего персонажа
2. **Перетащите предмет** из инвентаря на пустое место на сцене
3. **Запрос отправится** GM на создание лутбокса
4. **GM создаст** лутбокс от своего имени
5. **Вы получите** уведомление о создании

## 📋 **Подробная инструкция**

### **Шаг 1: Подготовка**
Убедитесь что:
- ✅ Treasure Hoard Manager загружен и активен
- ✅ Вы находитесь на сцене с активным слоем токенов
- ✅ У вас есть права на изменение сцены (для GM) или предметы в инвентаре (для игроков)

### **Шаг 2: Выбор предмета**
#### **Из компендиума:**
1. Нажмите на иконку компендиумов в правой панели
2. Выберите нужный компендиум (например, "Items")
3. Найдите предмет в списке
4. Нажмите и удерживайте левую кнопку мыши на предмете

#### **Из инвентаря:**
1. Откройте лист персонажа
2. Перейдите во вкладку "Inventory"
3. Найдите предмет в инвентаре
4. Нажмите и удерживайте левую кнопку мыши на предмете

### **Шаг 3: Перетаскивание на сцену**
1. **Перетащите предмет** из компендиума/инвентаря на сцену
2. **Наведите курсор** на пустое место (где нет других токенов)
3. **Отпустите кнопку** мыши

### **Шаг 4: Автоматическое создание**
Система автоматически:
- ✅ Проверит что под курсором нет других токенов
- ✅ Создаст актера для лутбоксов (если нужно)
- ✅ Создаст токен на сцене с правильными флагами
- ✅ Добавит предмет в инвентарь токена
- ✅ Покажет уведомление о создании

## 🔧 **Технические детали**

### **Что происходит при дропе:**
1. **Hook `dropCanvasData`** перехватывает событие
2. **Проверка типа** - только предметы (`type: "Item"`)
3. **Проверка позиции** - нет токенов под курсором
4. **Создание через сокет** - GM выполняет операцию
5. **Создание токена** с `actorLink: false`
6. **Добавление предмета** в актера токена
7. **Установка флагов** THM для контейнера

### **Настройки создаваемого лутбокса:**
```javascript
{
  name: "Название предмета",
  actorLink: false,  // Независимый инвентарь
  disposition: "neutral",
  displayName: "hover",
  flags: {
    "treasure-hoard-manager": {
      type: "container",
      enabled: true,
      data: {
        deleteWhenEmpty: true,
        showItemName: true,
        displayOne: true
      }
    }
  }
}
```

## 🛠️ **Устранение проблем**

### **Проблема: Ничего не происходит при дропе**
**Решения:**
- Проверьте что Treasure Hoard Manager загружен
- Убедитесь что вы на слое токенов
- Проверьте что под курсором нет других токенов
- Проверьте консоль на ошибки

### **Проблема: Предмет не создается**
**Решения:**
- Проверьте права доступа к сцене
- Убедитесь что предмет имеет тип "Item"
- Проверьте что socketlib работает
- Проверьте что вы GM (для прямого создания)

### **Проблема: Игрок не может создать лутбокс**
**Решения:**
- Убедитесь что GM онлайн
- Проверьте что socketlib работает
- Проверьте права игрока на инвентарь
- Проверьте консоль GM на сообщения об ошибках
